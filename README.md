# எண்மின் தோட்டம் 

## தமிழ்

இது **Next.js** பயன்படுத்தி உருவாக்கப்பட்ட, பின்நிலை(பேகென்டு) இல்லாத ஒரு வலைத்தளம்.

**எண்மின் தோட்டம்** என்பது ஒரு தனிப்பட்ட *டிஜிடல் கார்டன்* ஆகும்.  
தற்போது, இதில் **Markdown அடிப்படையிலான வலைப்பதிவுகள் (பிலாக்)** மட்டுமே உள்ளன.

---

# Yenmin Thottam (Digital Garden)

## English

This is a website built using **Next.js**, without any backend.

It is a small personal website that serves as a kind of **digital garden**—hence the name **எண்மின் தோட்டம்**, which translates to *Digital Garden*.  
For now, the site only contains **Markdown-based blogs**.
